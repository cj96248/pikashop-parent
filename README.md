# PiKa 商城

项目基于Dubbo开发的B2B2C网上商城，模拟京东商城的业务原型，分模块开发，主要分为网站前台、运营商后台、商家管理后台三个子系统。

## 技术栈

- DubboX
    - Zookeeper
- AngularJS
    - AdminLTE
    - pagination
    - select2
    - kindeditor
- Database
    - MySQL
    - MyBatis
